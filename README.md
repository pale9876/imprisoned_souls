# Imprisoned Souls


이 2D 프레임워크는 2D 액션 게임을 입체적으로 구현하기 위해 만든 오픈 프레임워크입니다. 
Server API를 활용하여 기존의 엔진 에디터에서 제공되지 않는 객체의 지원과 여러 최적화 효과를 쉽게 구현하기 위하여 만들고 있습니다.


이 프레임워크를 만든 이유는

- 저레벨 서버API로 구현할 수 있는 더 많은 기능을 더 유연하게 제공하고
- 노드의 생성, 이동, 할당, 해제 과정에서 발생하는 부하를 최대한 줄이고
- 사용자 친화적이고 툴 지원이 쉬운 에디터 환경을 구축

하기 위해서입니다.


이 프레임워크에는 다음과 같은 기능들을 포함하고 있습니다.
- 데모 게임 2종
- 액션게임을 위한 복잡한 인풋의 실시간 제어
- 하위 노드만을 대상으로 순환되어 작동되는 z축 소팅 및 드로우 콜 제어
- 부하 적은 2D 파티클 효과 구현
- 게임 내 이벤트를 실시간으로 전송하고 관리하는 싱글톤 객체들
- 가벼운 물리곡선 오브젝트 (예정)
- 클래스 제어와 디버깅을 위한 에디터 인스펙터 확장 툴 (예정)
