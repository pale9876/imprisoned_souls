# Imprisoned Souls


이 프레임워크는 게임이 아닙니다. 2D 액션 게임을 입체적으로 구현하기 위해 만든 엔진 프레임워크입니다.
Server API를 활용하여 기존의 엔진 에디터에서 제공되지 않는 객체의 지원과 여러 최적화 효과를 쉽게 구현하기 위하여 만들고 있습니다.


이 프레임워크에는 다음과 같은 기능들을 포함하고 있습니다.
- 데모 게임 2종
- 실시간 인풋 제어
- 씬 내부만을 대상으로 순환되어 작동되는 z축 소팅 및 드로우 콜 제어
- 커스텀 가능한 PhysicsServer2D 기반 파편효과
- 조인트와 리지드바디를 사용하지 않는 가벼운 물리적인 곡선 오브젝트
- 게임 내 이벤트를 실시간으로 전송하고 관리하는 싱글톤 객체들
- 클래스 제어와 디버깅을 위한 에디터 인스펙터 확장 툴
- 하나의 노드에 여러 Shader를 적용 가능


해당 프레임워크는 Z축을 가진 Node2D 객체인 EEAD2D가 Sprite2D 객체 대신 다양한 형태의 렌더링 기능을 수행할 수 있습니다.<br/><br/>
EEAD2D는 EndekaRenderer(엔데카 렌더러)를 통하여 Z축 렌더링이 수행됩니다. 없다면 씬트리에 기반하여 일반적인 렌더링을 수행합니다.<br/><br/>
EndekaRenderer는 EndekaRenderingItem의 z: float의 값을 기준으로 오름차순 정렬 후, 순차적으로 렌더링을 수행하고 z축이 가장 큰 정보를 마지막으로 렌더링합니다.<br/><br/>
